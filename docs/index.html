Prism.php (WIP)

This is a php port of prism.js syntax highlighter originally written by @LeaVerou.

Requirements

  • PHP >= 5.4
  • libxml >= 2.6.0

Installation

{
    "require" : {
        "vkbansal/prism" : "dev-master"
    }
}

Usage

<?php

use VKBansal\Prism\Prism;

$prism = new Prism();
$code = $prism->highlightHTML('<html>string</html>');
echo $code;
