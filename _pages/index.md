This is a php port of [prism.js](https://github.com/LeaVerou/prism) syntax highlighter originally written by [@LeaVerou](http://github.com/LeaVerou).

##Requirements
- PHP >= 5.4
- libxml >= 2.6.0

##Installation
```bash
composer require vkbansal/prism
```
